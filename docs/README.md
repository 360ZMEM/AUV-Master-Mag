## 🌟 核心要点回顾

本系统的核心思想可以概括为一句话：

> **AUV 不直接"看见"电缆，而是通过反复横切制造峰值观测，再把这些峰值观测拟合成中心线，并用置信度控制自己何时搜索、何时贴近、何时保持。**

三大支柱：
- **之字形搜索**：空间搜索转化为时间序列分析
- **磁感知**：从振荡、含噪信号中提取稳定强度
- **行为树**：优先级驱动的模块化决策架构
