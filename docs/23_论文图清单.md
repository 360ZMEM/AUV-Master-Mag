# 论文图清单（Paper Figure Index）

> **定位**：本文是 17–22 号文档与论文写作共用的图索引。可视化体系（参见 [10_可视化体系](10_可视化体系.md)）会同时输出"工程整体大图"（多面板 PNG）与"论文拆分小图"（单面板 PNG + PDF，IEEE 双栏单图宽度 ~3.5 in，字号 8–9 pt）。论文写作时直接引用本文档列出的 PNG/PDF 即可，无需再去翻阅整体图。

## 1. 输出位置与命名

`tools/visualize.py` 在执行时会产生如下两类输出：

| 类别 | 路径 | 格式 | 用途 |
| --- | --- | --- | --- |
| 工程整体大图 | `results/<ts>/<case>/figures/<case>_<tier>.png` | PNG | 投影、调试、报告 |
| 论文单面板小图 | `results/<ts>/<case>/figures/paper/<case>_<slug>.png` 与 `.pdf` | PNG + PDF | 论文正文 figure |
| Showcase 论文小图 | `results/<ts>/paper/showcase_<slug>.{png,pdf}` | PNG + PDF | 跨 case 总览 figure |
| Progress 论文小图 | `results/<ts>/paper/progress_<field>.{png,pdf}` | PNG + PDF | before/after 对比 figure |

默认开启论文小图；若需关闭可加 `--no-paper-figures`。

## 5. 论文小图风格（rcParams）

样式由 `_apply_paper_style()` 统一设置，关键参数：

| 项 | 值 | 用途 |
| --- | --- | --- |
| `font.family` | serif（Times New Roman / DejaVu Serif） | IEEE 风格 |
| `font.size` | 8.5 | 双栏单图基础字号 |
| `axes.titlesize` | 9.0 | 不再加 "(a)" 前缀，标题直接是物理含义 |
| `axes.labelsize` | 8.5 | x/y 轴名 |
| `xtick.labelsize / ytick.labelsize` | 7.5 | 刻度数字 |
| `axes.linewidth` | 0.8 | 边框细线 |
| `lines.linewidth` | 1.0 | 主曲线细线 |
| `lines.markersize` | 3.0 | 紧凑散点 |
| `legend.fontsize` | 6.8 | 图例 |
| `savefig.bbox` | `tight` | 自动裁白边 |

PNG 用于预览（dpi=300），PDF 用于论文 LaTeX 嵌入（vector）。

## 6. 推荐生成命令

```bash
# 1. 单 case 完整论文图（推荐 case1, case_maze_sonar_dropout）
python tools/visualize.py --case case1
python tools/visualize.py --case case_maze_sonar_dropout

# 2. 全场景 + showcase + paper 拆图
python tools/visualize.py --all

# 3. case1v..6v zig-zag 变体回归
python tools/visualize.py --variants

# 4. maze 系列鲁棒性
python tools/visualize.py --maze

# 5. before/after 论文 ablation
python tools/visualize.py --progress

# 6. 仅需大图、不需论文小图
python tools/visualize.py --all --no-paper-figures
```

## 7. 与 17–22 号文档的交叉引用

各文档"## 关键可视化"小节会指向本文档的具体 slug，以便定位"这一段叙述对应哪张论文图"。建议引用约定：

```text
（详见 [23_论文图清单 §2.2 detail_probe_cycle](23_论文图清单.md#22-detail-面板12-张配合-181920-号文档)）
```

或直接给图名 `detail_probe_cycle` 即可。

## 8. 局限与注意事项

1. 单面板小图与整体大图共用 painter，因此**面板内容**保持一致，但论文小图重新调整了字号、线粗、边距。
2. PDF 体积一般 < 25 KB（矢量），PNG 体积 ~30–60 KB（dpi=300）。如果论文模板限制 PDF 体积，可降 dpi。
3. 若论文需要灰度打印，请在审稿后端额外做颜色映射；本系统当前保持冷/暖（声呐/磁）语义色，不强制灰度。
4. `--no-paper-figures` 仅关闭论文小图，不影响大图。
