# 磁感知方法论

## 1. 从物理场到导航量

### 1.1 感知链路的复杂性

磁力计读到的不是"电缆距离"，而是三轴磁场分量。要把它变成可用于控制的"跟踪强度"和"中心线方向"，中间要经过多层处理：

```
电缆电流 → 物理磁场 → 传感器测量 → 坐标变换 → 信号处理 → 特征提取 → 状态估计 → 导航决策
   ↓          ↓           ↓           ↓           ↓           ↓           ↓           ↓
 Biot-Savart  背景场叠加    噪声干扰      姿态补偿      带通滤波      峰值检测      置信度融合    航向指令
              地磁25μT     量化误差      安装矩阵      RMS提取       线拟合        SNR评估      速度调节
              日变干扰     零偏漂移      Body→NED     包络平滑      历史管理      残差分析     Zig-zag
                          非正交        带通提取      中值滤波      异常检测                    螺旋搜索
```

### 1.2 核心挑战

| 挑战 | 物理本质 | 应对策略 |
|------|----------|----------|
| **工频振荡** | 50Hz 信号瞬时值正负变化 | RMS 提取、包络平滑 |
| **姿态干扰** | 滚转/俯仰改变测量分量 | 坐标系变换、姿态补偿 |
| **噪声污染** | 手机级噪声可达 150nT | 带通滤波、中值滤波、迟滞 |
| **背景场** | 地磁 25-50μT 远大于信号 | 异常提取、背景扣除 |
| **量化误差** | 低比特率引入量化噪声 | 高保真模式模拟、插值 |

## 2. 坐标系变换链

### 2.1 三层坐标系

```
┌─────────────────────────────────────────────────────────┐
│ NED 系（北东地）                                          │
│ - 北向轴：地理北方                                        │
│ - 东向轴：地理东方                                        │
│ - 地向轴：向下（重力方向）                                 │
│ - 作用：全局导航、异常提取                                 │
└─────────────────┬───────────────────────────────────────┘
                  │ Body → NED 旋转矩阵（IMU 姿态）
┌─────────────────▼───────────────────────────────────────┐
│ Body 系（机体系）                                         │
│ - X 轴：AUV 纵轴（向前）                                  │
│ - Y 轴：AUV 横轴（向右）                                  │
│ - Z 轴：AUV 垂轴（向下）                                  │
│ - 作用：运动控制、航向指令                                 │
└─────────────────┬───────────────────────────────────────┘
                  │ Sensor → Body 旋转矩阵（安装误差）
┌─────────────────▼───────────────────────────────────────┐
│ Sensor 系（传感器系）                                     │
│ - 三轴磁力计的原始坐标系                                  │
│ - 可能存在安装误差、非正交                                 │
│ - 作用：原始测量                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 旋转变换公式

**Sensor → Body**（静态安装矩阵）：

$$
\mathbf{B}_{body} = \mathbf{R}_{sensor \to body}(\phi, \theta, \psi) \cdot \mathbf{B}_{sensor}
$$

其中 $(\phi, \theta, \psi)$ 是安装误差的欧拉角（典型值 < 1°）。

**Body → NED**（IMU 姿态）：

$$
\mathbf{B}_{NED} = \mathbf{R}_{body \to NED}(r, p, h) \cdot \mathbf{B}_{body}
$$

其中 $(r, p, h)$ 是实时测量的滚转、俯仰、航向角。

**完整的两步变换**：

$$
\mathbf{B}_{NED} = \mathbf{R}_{body \to NED} \cdot \mathbf{R}_{sensor \to body} \cdot \mathbf{B}_{sensor}
$$

### 2.3 为什么需要两步变换？

1. **静态误差**：安装矩阵是固定的，可以在实验室标定
2. **动态误差**：IMU 姿态是实时变化的，必须在线补偿
3. **解耦设计**：分别处理静态误差和动态变化

### 2.4 姿态扰动的验证（case4）

场景：AUV 经历 ±10° 滚转、±5° 俯仰的周期性扰动

**如果不补偿**：
- 测量值会随姿态大幅振荡
- 峰值检测失效
- 拟合结果混乱

**补偿后**：
- NED 系下的异常场保持稳定
- 峰值检测正常
- 跟踪性能不受影响

## 3. AC 信号的带通与 RMS 提取

### 3.1 为什么不能直接用瞬时值？

对于 50Hz 工频 AC 信号：

$$I(t) = I_0 \sin(2\pi f t + \phi)$$

磁场与电流成正比：

$$B(t) \propto I(t)$$

**直接用瞬时值的问题**：

1. **正负半周互相抵消**：+100nT 和 -100nT 的平均值是 0
2. **采样时刻敏感**：不同时刻采样得到完全不同的值
3. **噪声放大**：瞬时值对宽带噪声极其敏感

### 3.2 带通滤波的设计

**目标**：只保留目标频段（50Hz ± 8Hz），剔除其他频率

**实现**：二阶 Butterworth 带通滤波器

$$
H(s) = \frac{s \cdot \omega_0 / Q}{s^2 + s \cdot \omega_0 / Q + \omega_0^2}
$$

其中：
- $\omega_0 = 2\pi f_0$ 是中心频率（50Hz）
- $Q = f_0 / BW$ 是品质因数
- $BW$ 是带宽（典型 16Hz，即 ±8Hz）

**离散化**：使用双线性变换得到数字滤波器系数（SOS 形式）

**流式处理**：滤波器状态在采样块之间保持

### 3.3 RMS 提取

**目的**：将 AC 信号转换为"强度"度量

**定义**：均方根值

$$
\text{RMS} = \sqrt{\frac{1}{N}\sum_{i=1}^N x_i^2}
$$

对于正弦波，RMS = 峰值 / $\sqrt{2}$

**滑动窗口 RMS**：

1. 窗口大小：至少包含 2 个完整周期
   - 50Hz → 2 周期 = 40ms → 200Hz 采样率下 8 个样本
2. 滑动更新：每个新样本更新 RMS 值
3. 物理意义：目标频段的"能量强度"

### 3.4 包络平滑

RMS 值本身仍有波动，需要进一步平滑：

**一阶低通滤波器**：

$$
y[n] = (1 - \alpha) \cdot y[n-1] + \alpha \cdot x[n]
$$

其中 $\alpha = \frac{\Delta t}{\tau + \Delta t}$，$\tau$ 是时间常数（典型 0.3-0.5s）

**效果**：
- 保留信号的空间变化趋势
- 抑制高频波动
- 产生可用于控制的平滑包络

### 3.5 完整的 AC 处理链路

```
原始三轴信号 (200Hz)
    ↓
坐标变换 (Sensor → Body → NED)
    ↓
背景场扣除 (减去地磁)
    ↓
带通滤波 (50Hz ± 8Hz)
    ↓
中值滤波 (窗口 5-7) ← 压除脉冲噪声
    ↓
带通信号 RMS (滑动窗口，≥2 周期)
    ↓
低通平滑 (时间常数 0.3-0.5s)
    ↓
跟踪强度 (可用于控制)
```

## 4. DC 信号的简单处理

对于 DC 模式（稳恒电流），处理链路更简单：

```
原始三轴信号
    ↓
坐标变换 (Sensor → Body → NED)
    ↓
背景场扣除
    ↓
向量范数 (magnitude = √(Bx² + By² + Bz²))
    ↓
低通平滑
    ↓
跟踪强度
```

**关键区别**：不需要带通和 RMS，直接用幅值。

## 5. 信噪比（SNR）估计

### 5.1 定义

$$
\text{SNR}_{linear} = \frac{S_{signal}}{S_{noise}}
$$

$$
\text{SNR}_{dB} = 20 \log_{10}(\text{SNR}_{linear})
$$

### 5.2 噪声底估计

**方法 1**：基于已知传感器噪声

$$N_{floor} = \sigma_{sensor}$$

**方法 2**：基于信号波动（自适应）

$$N_{floor} = \text{median}(|B(t) - B_{smooth}(t)|)
$$

**方法 3**：频域估计（AC 模式）

1. 计算 FFT
2. 目标频段外的频谱能量作为噪声估计

### 5.3 可靠性判断

| SNR (dB) | 可靠性 | 行为 |
|----------|--------|------|
| < 6 | 不可靠 | 降低置信度、搜索模式 |
| 6-12 | 弱信号 | 谨慎跟踪、降低宽度 |
| 12-20 | 中等信号 | 正常跟踪 |
| > 20 | 强信号 | 高置信度、贴缆模式 |

## 6. 峰值检测的细节

### 6.1 三状态机

```
┌──────────┐
│   IDLE   │ ← 等待上升沿
└────┬─────┘
     │ 上升≥N次
     ↓
┌──────────┐
│ASCENDING │ ← 记录峰值候选
└────┬─────┘
     │ 停止上升
     ↓
┌──────────┐
│ PEAK_ZONE│ ← 寻找最佳峰值
└────┬─────┘
     │ 下降≥M次 + 迟滞
     ↓
  EMIT_PEAK
     │
     ↓ 冷却期
┌──────────┐
│   IDLE   │
└──────────┘
```

### 6.2 迟滞的作用

防止在"平台期"振荡：

```
强度
  │
  │     ╱────╲
  │    ╱      ╲
  │   ╱        ╲
  │  ╱          ╲
  │ ╱            ╲
  └────────────────────
      ↑     ↑     ↑
    触发  迟滞  确认
    上升  带    下降
```

**迟滞带**：峰值 × (1 - hysteresis_fraction)

典型值：8% 迟滞，即峰值下降到 92% 时才确认过峰。

### 6.3 冷却期的作用

防止在一次穿缆中触发多个峰值：

- 典型冷却期：0.5-1.0 秒
- 物理意义：AUV 离开峰值区域的过渡时间
- 效果：确保每次 Zig-zag 半周期只触发一次过峰

## 7. 高保真磁力计模拟

### 7.1 标准模式 vs 高保真模式

| 特性 | 标准模式 | 高保真-手机 | 高保真-工业 |
|------|----------|-------------|-------------|
| **白噪声** | 0.05 nT | 0.03 nT | 0.03 nT |
| **粉红噪声** | 无 | 0.08 nT | 0.08 nT |
| **脉冲噪声** | 无 | 2% 概率 | 2% 概率 |
| **AUV 干扰** | 无 | 35/-20/15 nT | 35/-20/15 nT |
| **采样率** | 200 Hz | 200 Hz | 1000 Hz |
| **比特深度** | 理想 | 24-bit | 24-bit |

### 7.2 粉红噪声（1/f 噪声）

**特点**：功率谱密度与频率成反比

$$S(f) \propto \frac{1}{f}$$

**影响**：
- 低频成分更强
- 导致长期漂移
- 更难滤除

**模拟方法**：
1. 生成白噪声
2. 在频域乘以 $\sqrt{1/f}$
3. 逆变换回时域

### 7.3 脉冲噪声

**模型**：随机时刻的脉冲

$$
n_{impulse}(t) = \sum_{i} A_i \cdot e^{-(t - t_i)/\tau_{decay}} \cdot u(t - t_i)
$$

其中：
- $t_i$：脉冲时刻（2% 概率/样本）
- $A_i$：脉冲幅度（60 nT）
- $\tau_{decay}$：衰减时间常数（8 样本）

### 7.4 AUV 静态干扰

**来源**：AUV 自身的电子设备、电机

**模拟**：在机体系下添加常数偏置

$$\mathbf{B}_{interference, body} = [35, -20, 15] \text{ nT}$$

经过姿态变换后，在 NED 系下表现为周期性波动。

### 7.5 量化噪声

**低比特率传感器的影响**：

$$B_{quantized} = \text{round}\left(\frac{B}{\Delta}\right) \cdot \Delta$$

其中 $\Delta = \frac{R_{fullscale}}{2^N - 1}$ 是量化步长。

对于 24-bit、±100k nT 量程：
- $\Delta \approx 0.012$ nT（可忽略）

对于 12-bit、±100k nT 量程：
- $\Delta \approx 24$ nT（显著噪声）

## 8. 感知增强层

### 8.1 包络梯度跟踪

**目的**：从 RMS 包络的空间变化推断电缆方向

**方法**：Savitzky-Golay 滤波器计算梯度

1. 对包络信号应用 SG 滤波（导数模式）
2. 得到 $\frac{dS}{dt}$（时间梯度）
3. 除以速度得到 $\frac{dS}{dx}$（空间梯度）

$$
\nabla S = \frac{dS/dt}{v_{vehicle}}
$$

**物理含义**：
- 正梯度：信号增强方向 → 电缆在此方向
- 负梯度：信号减弱方向 → 远离电缆
- 梯度大小：信号变化速率

**用途**：
- 部署模式下的方向估计
- 与拟合结果交叉验证
- 安全锁准则 B 的输入

### 8.2 磁场矢量分析

**物理约束**：在穿缆点，水平磁场矢量 $\mathbf{B}_{xy}$ 与电缆方向垂直

**方法**：
1. 提取 NED 系下的水平磁场 $(B_x, B_y)$
2. 计算矢量方向：$\psi_{vector} = \arctan2(B_y, B_x)$
3. 电缆方向：$\psi_{cable} = \psi_{vector} + 90°$

**置信度**：基于矢量的一致性（圆形统计）

$$
C = \sqrt{(\frac{1}{N}\sum \sin\psi_i)^2 + (\frac{1}{N}\sum \cos\psi_i)^2}
$$

$C = 1$：完全一致；$C = 0$：完全随机

**用途**：
- 独立于峰值拟合的航向估计
- 部署模式下的初始化
- 多传感器融合

## 9. 置信度融合

### 9.1 磁置信度

融合三个因素：

$$
C_{mag} = 0.45 \cdot C_{SNR} + 0.35 \cdot C_{fit} + 0.20 \cdot C_{age}
$$

其中：
- $C_{SNR}$：基于信噪比的置信度
- $C_{fit}$：基于拟合残差的置信度
- $C_{age}$：基于检测年龄的置信度（指数衰减）

### 9.2 融合置信度

根据导航源不同，采用不同策略：

| 导航源 | 融合公式 |
|--------|----------|
| SONAR | $0.8 C_{sonar} + 0.2 C_{mag}$ |
| MAGNETIC | $0.8 C_{mag} + 0.2 C_{sonar}$ |
| MEMORY | $0.24 + 0.52 C_{mag} + 0.14 C_{fit}$ |
| BLIND | $\min(0.4, 0.6 C_{mag} + 0.4 C_{sonar})$ |

### 9.3 置信度门限

| 置信度范围 | 模式 | 行为 |
|------------|------|------|
| < 0.3 | SEARCH | 大宽度 Zig-zag |
| 0.3-0.7 | APPROACH | 中等宽度跟踪 |
| > 0.7 | HOLD | 小宽度或贴缆 |

## 10. 性能优化

### 10.1 计算优化

1. **缓存 FFT 窗函数**：避免重复计算
2. **复用滤波器状态**：在采样块之间保持
3. **稀疏更新**：某些计算不需要每个控制步都执行

### 10.2 内存优化

1. **固定容量队列**：使用 deque
2. **及时释放**：不需要的历史数据丢弃
3. **视图而非拷贝**：大数组使用 NumPy 视图

### 10.3 实时性保证

- 控制循环：20Hz（50ms）
- 感知处理：< 10ms
- 滤波和拟合：< 5ms
- **总计**：< 20ms，满足实时要求

## 11. 总结

磁感知方法论的三大支柱：

1. **坐标变换**：确保测量反映真实空间关系
2. **信号处理**：从振荡、含噪信号中提取稳定强度
3. **状态估计**：从峰值序列中拟合电缆方向

**核心公式**：

$$
\text{可靠感知} = \text{正确坐标变换} \times \text{有效信号处理} \times \text{鲁棒状态估计}
$$
