# AUV 磁电缆跟踪仿真实验室 - 系统概述

## 1. 系统定位

本项目是一个基于洋葱架构的 AUV（自主式水下航行器）磁电缆跟踪数学仿真实验室 Phase 1 实现。其核心目标不是复刻完整的海上测绘系统，而是将"数学灵魂"跑通——在一个可调参、可扩展的 Python 工程中，将磁场生成、坐标补偿、传感器误差、过峰检测、中心线拟合、Zig-zag 控制和实时可视化闭环联通。

### 1.1 核心问题域

海底电力电缆磁跟踪本质上是一个**连续闭环估计问题**，而非单次检测问题：

- **单次测量**只能回答："这里的磁异常强不强？"
- **连续测量**才能逐步回答："电缆在哪里？朝哪个方向延伸？我在左侧还是右侧？"

### 1.2 方法论选择：之字形搜索

为什么采用之字形（Zig-zag）搜索而非直接追踪最大信号？

1. **磁场是空间分布场**，不是单点目标
2. **工频 AC 磁场振荡**导致瞬时值不稳定
3. **姿态变化**会影响测量值
4. **直接追逐最大值**容易在噪声中抖动

之字形搜索的本质是：通过反复横切制造"穿缆观测"，将多个峰值点拟合成中心线，从而估计电缆方向。

## 3. 核心技术特点

### 3.1 块采样机制（Block Sampling）

**问题**：50Hz 工频信号若只在 20Hz 控制循环上取单点，违反采样定理。

**解决方案**：
- 外层控制：20Hz（dt_s=0.05s）
- 内层采样：200Hz（每控制步 10 个样本）
- 感知处理：在高采样块上完成带通、RMS、包络更新

这是 AC 感知能够成立的物理基础。

### 3.2 坐标系变换链

```
地磁背景场 (NED) → 传感器坐标系 (Sensor) → 机体系 (Body) → NED 系
      ↑                  ↑                    ↑            ↑
   真值参考          安装矩阵            姿态补偿      异常提取
```

关键变换：
- **Sensor → Body**：静态安装矩阵（处理安装误差）
- **Body → NED**：IMU 姿态（处理滚转、俯仰、航向）

### 3.3 多模态感知融合

| 传感器 | 作用范围 | 可靠性 | 使用场景 |
|--------|----------|--------|----------|
| 磁力计 | 近距离（<15m） | 信号依赖 | 峰值检测、穿缆定位 |
| 声纳 | 中远距离（可达30m） | 环境依赖 | 远距离导引、信号弱时辅助 |
| 记忆（Memory） | 短时（<4s） | 衰减依赖 | 临时信号丢失时保持 |

融合策略：
- SONAR：距离≥7.5m 或信号弱时优先
- MAGNETIC：信号强且在最优范围时优先
- MEMORY：检测超时但残差小时启用
- BLIND：无可观测信息时的盲从导航

### 3.4 三态任务状态机

```
                    ┌─────────────────┐
                    │  SEARCH_ZIGZAG  │ 沿先验走廊横扫，速度 1.0
                    └────────┬────────┘
                             │ 连续检测到信号（迟滞锁存）
                    ┌────────▼────────┐
                    │   LOCK_ALIGN    │ 减速 0.5，等待拟合收敛
                    └────────┬────────┘
                             │ 拟合收敛（P_yy 代理 + 偏航 + 置信度）
                    ┌────────▼────────┐
                    │  TRACK_ACTIVE   │ 声呐/磁协同稳态贴缆，速度 1.0
                    └────────┬────────┘
                             │ 置信度持续崩溃
                    ┌────────▼────────┐
                    │EMERGENCY_SURFACE│ 双盲终态，速度 0 + 上浮标志
                    └─────────────────┘
```

转移要点（详见 [`03_任务状态机方法论.md`](03_任务状态机方法论.md)）：
- **信号判定迟滞锁存**：上升沿 50 nT 触发、下降沿 25 nT 才解除，避免 zig-zag 峰谷振荡反复翻转。
- **丢失判定按时效**：持续 `signal_hold_s`（~12 s，覆盖一个完整扫描周期）无新检测才判丢失。
- **转移不对称**：TRACK_ACTIVE 信号丢失退回 LOCK_ALIGN（非 SEARCH），保留已收敛的拟合。
- **所有阈值集中在 `MissionThresholds`**，便于人工修改。

## 4. 关键创新点

### 4.1 感知锚定安全锁（Perception-Anchored Safe-Lock）

**问题**：信号丢失后，系统可能跟踪陈旧的拟合结果。

**现行机制**：峰值跌落安全锁——当确认峰值强度较上一次确认值跌落超过 `safe_lock_peak_drop_nt` 时，短时锁定（`safe_lock_until_s`），抑制对陈旧信号的横向压线。

> 早期还设计过两条增强准则（准则 A：强度崩溃 + 过度位移使拟合失效；准则 B：梯度方向与拟合线法线不一致降置信度）。声呐定向使拟合无歧义后二者已停用（`safe_lock_criterion_a/b_active` 恒为 False，仅作惰性诊断字段保留）。

### 4.2 声呐定向的无歧义中心线拟合

**问题**：单纯磁穿缆只能得到 "航向 ± 90°" 两个候选，早期版本用奇偶分组 + 角度扩散最小化做 ±90° 消歧，逻辑繁琐且易抖。

**现行方案**：声呐位置直接喂入加权 PCA 线拟合器（`WeightedSlidingWindowFitter`），拟合方向天然无歧义——旧的 ±90° 消歧机制已整体删除，部署模式下航向估计直接镜像拟合中心线方向。

### 4.3 包络梯度跟踪（Envelope Gradient Tracking）

**目的**：部署模式下增强弱信号时的方向估计。

**方法**：
1. 使用 Savitzky-Golay 滤波器计算包络梯度
2. 将时间梯度转换为空间梯度（nT/m）
3. 梯度方向指示信号增强方向 → 电缆方向

### 4.4 磁场矢量分析（Magnetic Vector Analysis）

**物理约束**：在电缆穿缆点（峰值），水平磁场矢量 B_xy 与电缆方向垂直。

**用途**：
- 提供独立的电缆航向估计
- 与拟合航向交叉验证
- 增强部署模式下的方向感知

## 5. 系统模式

### 5.1 信号模式

| 模式 | 频率 | 处理链路 | 适用场景 |
|------|------|----------|----------|
| DC | 0 Hz | 幅值直接跟踪 | 稳恒电流测试 |
| AC 50Hz | 50 Hz | 带通 → RMS → 包络 | 标准工频（中国/欧洲） |
| AC Demo | 10-20 Hz | 带通 → RMS → 包络 | 实验对照 |

### 5.2 部署模式

| 模式 | 名义路由先验 | 搜索策略 | 适用场景 |
|------|--------------|----------|----------|
| 标准模式 | 启用 | Zig-zag | 已知电缆路由的大概走向 |
| 部署模式 | 禁用 | 螺旋搜索 + 盲从 | 未知电缆路由的首次探测 |

### 5.3 磁力计模式

| 模式 | 噪声水平 | 采样率 | 量程 | 典型应用 |
|------|----------|--------|------|----------|
| 标准模式 | 0.05 nT | 200 Hz | 100k nT | 算法验证 |
| 高保真-手机 | 150 nT + 粉红噪声 | 200 Hz | 100k nT | 消费级设备测试 |
| 高保真-工业 | 0.03 nT + 粉红噪声 | 1000 Hz | 100k nT | 工业级设备模拟 |

## 7. 性能指标

### 7.1 跟踪性能

- **航向误差**：< 5°（理想条件）
- **侧向偏差**：< 2m（稳态跟踪）
- **沿轨覆盖率**：> 0.85（部署模式）
- **峰值捕获率**：> 0.90（标准场景）

### 7.2 鲁棒性

- **SNR 阈值**：6 dB（可靠检测）
- **丢失恢复**：TRACK_ACTIVE 信号丢失退回 LOCK_ALIGN，按 `signal_hold_s`（~12 s）时效判定
- **转弯响应延迟**：< 2 次穿缆（case2）
